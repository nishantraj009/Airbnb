# Airbnb
This project is a simple Airbnb-style website made using HTML and CSS. It shows property listings with images, prices, ratings, and short details. The design is clean and responsive, so it works well on both mobile and desktop screens. This project helps in understanding layout design, cards, navigation bar, and basic styling using CSS.
